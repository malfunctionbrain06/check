QUICK START (5 MINUTES)
================================================================================

STEP 1: Setup MongoDB Atlas
─────────────────────────────
1. Go to https://www.mongodb.com/cloud/atlas
2. Create free account and project
3. Create M0 (free) cluster
4. Create database user with password
5. Whitelist your IP (allow from anywhere for dev)
6. Copy connection string

STEP 2: Create .env.local
──────────────────────────
Create file: /vercel/share/v0-project/.env.local

Add this line with your MongoDB connection string:
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/gulabi-guiltz?retryWrites=true&w=majority

STEP 3: Install Dependencies
──────────────────────────────
npm install
# or
pnpm install

STEP 4: Start Development Server
─────────────────────────────────
npm run dev
# or
pnpm dev

Visit http://localhost:3000

STEP 5: Add Your First Product
───────────────────────────────
Use this cURL command:

curl -X POST http://localhost:3000/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Mint Butterfly Clip",
    "code": "GG-001",
    "price": "₹299",
    "description": "A soft handcrafted butterfly clip made with love and precision.",
    "image": "https://images.unsplash.com/photo-1599643478518-a784e5dc4c8f?w=500&h=500&fit=crop",
    "category": "Hair Accessories"
  }'

STEP 6: View Products
──────────────────────
Visit http://localhost:3000/products

Your product should appear automatically!

================================================================================
API ENDPOINTS
================================================================================

GET /api/products
─────────────────
Fetch all products from database

Response (200 OK):
{
  "success": true,
  "data": [
    {
      "_id": "507f...",
      "name": "Mint Butterfly Clip",
      "code": "GG-001",
      "price": "₹299",
      "description": "...",
      "image": "https://...",
      "category": "Hair Accessories",
      "createdAt": "2024-01-15T10:30:00Z",
      "updatedAt": "2024-01-15T10:30:00Z"
    }
  ]
}


POST /api/products
──────────────────
Create a new product

Request Body:
{
  "name": "Product Name",
  "code": "GG-XXX",
  "price": "₹299",
  "description": "At least 10 characters, max 500",
  "image": "https://example.com/image.jpg",
  "category": "Optional category"
}

Response (201 Created):
{
  "success": true,
  "data": {
    "_id": "507f...",
    "name": "Product Name",
    ... (full product object)
  }
}

Response (400 Bad Request - Validation Error):
{
  "success": false,
  "error": "Validation failed",
  "details": {
    "errors": [
      {
        "message": "Description must be at least 10 characters",
        "path": ["description"]
      }
    ]
  }
}

Response (409 Conflict - Duplicate Code):
{
  "success": false,
  "error": "Product with this code already exists"
}

Response (500 Server Error):
{
  "success": false,
  "error": "Failed to create product"
}
