QUICK START GUIDE
================================================================================

STEP 1: Setup MongoDB (5 minutes)
  1. Go to https://www.mongodb.com/cloud/atlas
  2. Create free account
  3. Create M0 (free) cluster
  4. Create database user
  5. Get connection string

STEP 2: Configure Environment (1 minute)
  1. Create .env.local in project root
  2. Add MONGODB_URI with your connection string

STEP 3: Start Development Server (1 minute)
  npm install
  npm run dev

STEP 4: Add First Product (1 minute)
  curl -X POST http://localhost:3000/api/products \
    -H "Content-Type: application/json" \
    -d '{"name":"...","code":"...","price":"...","description":"...","image":"...","category":"..."}'

STEP 5: View on Website (1 minute)
  Visit http://localhost:3000/products

TOTAL TIME: ~10 minutes from start to first product

VALIDATION RULES
================================================================================

Field        | Type    | Min | Max | Required
─────────────┼─────────┼─────┼─────┼──────────
name         | string  | 1   | 100 | ✅ Yes
code         | string  | 1   | 50  | ✅ Yes (unique)
price        | string  | -   | -   | ✅ Yes
description  | string  | 10  | 500 | ✅ Yes
image        | string  | -   | -   | ✅ Yes (valid URL)
category     | string  | -   | 50  | ❌ No

All validation happens at both:
  1. Zod validation (API input)
  2. Mongoose schema validation (database)

================================================================================
API ENDPOINTS
================================================================================

GET /api/products
  ├─ Description: Fetch all products from database
  ├─ Method: GET
  ├─ Response: Array of products
  ├─ Status: 200 (success) or 500 (error)
  └─ Example: curl http://localhost:3000/api/products

POST /api/products
  ├─ Description: Create new product
  ├─ Method: POST
  ├─ Body: Product data (JSON)
  ├─ Response: Created product
  ├─ Status: 201 (created), 400 (invalid), 409 (duplicate), 500 (error)
  └─ Example: curl -X POST http://localhost:3000/api/products -H "Content-Type: application/json" -d '{"name":"...","code":"...","price":"...","description":"...","image":"...","category":"..."}'
